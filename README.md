# Diabetes Data Analysis Project
This project demonstrates data cleaning, feature engineering, visualization, and basic machine learning analysis using a diabetes medical dataset.
## Project structure
### Project goals
Explore a medical dataset to identify factors associated with diabetes and visualize relationships between health indicators.
### Data cleaning
Handling unrealistic or missing values in the dataset and preparing the data for analysis.
### Feature engineering
Creating age group categories to better analyze diabetes distribution across different age ranges.
### Data scaling
Applying StandardScaler to normalize numerical variables before analysis.
### Exploratory data analysis
Analyzing distributions and relationships between medical indicators and diabetes outcomes.
### Feature importance
Using a Random Forest model to identify which medical indicators are most influential in predicting diabetes.
